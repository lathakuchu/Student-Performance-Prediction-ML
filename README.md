# Student Performance Prediction Using Machine Learning

## Project Overview

This project predicts student marks using Machine Learning techniques. The model analyzes factors such as study hours, attendance, previous academic performance, assignments completed, and sleep hours to estimate a student's expected marks.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Joblib

## Machine Learning Algorithm

Linear Regression

## Features

- Study Hours
- Attendance Percentage
- Previous Marks
- Assignments Completed
- Sleep Hours

## Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis
4. Feature Selection
5. Train-Test Split
6. Model Training
7. Prediction
8. Model Evaluation
9. Visualization
10. Model Saving

## Results

The model successfully predicts student marks based on academic and behavioral factors.

### Evaluation Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score
